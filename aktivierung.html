E-Mail-Benachrichtigung freischalten

Diese Seite brauchst du nur ein einziges Mal.

  1. Unten auf den Button klicken. Damit geht eine Testanfrage raus.
  2. Es öffnet sich eine Seite von FormSubmit. Dort steht ein Bestätigungslink oder ein Code für ronald@tsa-connect.de. Folge den Anweisungen auf dieser Seite.
  3. Schau parallel in dein Postfach, auch in den Spam-Ordner. Absender ist FormSubmit.
  4. Sobald bestätigt: Formular auf der Webseite testen. Ab jetzt kommt jede Anfrage per Mail bei dir an.

Nach erfolgreicher Freischaltung kann diese Datei gelöscht werden. Sie ist nirgends auf der Webseite verlinkt und wird von Suchmaschinen ignoriert.
