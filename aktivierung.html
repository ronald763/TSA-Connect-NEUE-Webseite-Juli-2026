E-Mail-Benachrichtigung freischalten

Diese Seite brauchst du nur, bis die Benachrichtigung läuft.

Zuerst kurz prüfen Schick von einer beliebigen anderen Adresse (Handy, Gmail, GMX) eine Mail an ronald@tsa-connect.de. Kommt sie nicht an, existiert das Postfach noch nicht. Dann kann auch keine Bestätigung ankommen, und du musst das Postfach erst bei deinem Domain-Anbieter anlegen.
  1. Trag unten die Adresse ein, an die die Benachrichtigungen gehen sollen. Tipp für den Test: Nimm zuerst eine Adresse, von der du sicher weißt, dass sie funktioniert (z. B. deine Gmail-Adresse). So siehst du sofort, ob die Strecke insgesamt läuft.
  2. Auf Jetzt freischalten klicken. Es geht eine Testanfrage raus.
  3. Es öffnet sich eine Seite von FormSubmit. Lies genau, was dort steht, und folge den Anweisungen.
  4. Postfach prüfen, auch den Spam-Ordner. Absender ist FormSubmit.

Nach erfolgreicher Freischaltung kann diese Datei gelöscht werden. Sie ist nirgends auf der Webseite verlinkt und wird von Suchmaschinen ignoriert.
